# Pandas Data Manipulation and Visualization

## Overview
This repository contains Python scripts demonstrating core data analysis and manipulation techniques using Pandas.

## Topics Covered
- Reading and exploring datasets
- Indexing, filtering, and ordering data
- Merging, joining, and concatenating datasets
- GroupBy operations and data aggregation
- Data visualization

## Libraries Used
- Pandas
- NumPy
- Matplotlib

## Purpose
This project showcases practical Pandas operations commonly used in data analytics workflows.
